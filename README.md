## Installation Instructions

**System Requirements**

* `node 18`
* `npm 8`

**Clone and install**

```bash
git clone https://github.com/push-based/ws_enterjs_advanced_day_23.git

cd ws_enterjs_advanced_day_23
npm install

# (optional) if the step before didn't work, please try the following
npm install --force
```

**Run the application**

```bash
npm run start
```
